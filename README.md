# Baby Photoshop Program

A C++ command-line image processing application that allows users to load, modify, and save images. 

## Features
The program provides a wide variety of image filters and transformations:
- **Black and White Filter**
- **Flip Image** (Horizontal and Vertical)
- **Crop Image**
- **Grayscale**
- **Invert Colors**
- **Resize Image**
- **Purple Scale**
- **Rotate Image** (90, 180, 270 degrees)
- **Blur Image** (Weak, Normal, Strong)
- **Add Frame** (Normal or Fancy)
- **Infrared Effect**
- **Darken or Lighten Image**
- **Merge Images**
- **Edge Detection**
- **Oil Painting Effect**

## Usage
Run the compiled executable. You will be prompted with a menu to:
1. Load a new image
2. Apply filters
3. Save current image
4. Exit
